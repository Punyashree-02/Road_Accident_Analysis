# 🚦 Road Accident Analysis & Prediction System

## 📌 Overview

This project analyzes road accident data and predicts accident severity using Machine Learning.

## ⚙️ Technologies Used

* **Python** (Pandas, NumPy, Scikit-learn, Matplotlib) → Data preprocessing, analysis, and machine learning
* **Jupyter Notebook** → Model development and experimentation
* **Power BI** → Interactive dashboard and data visualization
* **SQL Server** → Data validation, querying, and structured data management
* **Microsoft Excel** → Pivot tables and preliminary data analysis


## 🤖 Machine Learning

* Model: Random Forest Regressor
* Clustering: KMeans (Risk Zones)
* Evaluation: MAE, R² Score

## 📊 Features

* Accident trend analysis
* Weather & vehicle impact
* Risk zone identification
* Casualty prediction

## 📁 Files

* `Road_accident.ipynb` → ML model
* `dashboard.pbix` → Power BI dashboard

## 🎯 Outcome

This project successfully combines data analytics and machine learning to analyze and predict road accident patterns.

🔹 Using Python, we:

* Performed data preprocessing and feature engineering
* Built a Random Forest model to predict accident severity (number of casualties)
* Applied KMeans clustering to identify high-risk accident zones
* Evaluated model performance using MAE and R² score

🔹 Using Power BI, we:

* Developed an interactive dashboard to visualize accident trends
* Analyzed casualties based on vehicle type, road type, and weather conditions
* Identified high-risk regions and monthly accident patterns

📊 Overall, the system helps in:

* Understanding accident trends
* Identifying high-risk areas
* Supporting data-driven decision-making for road safety improvement
